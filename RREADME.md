# Customer Churn Prediction

## Project Overview

This project predicts whether a telecom customer is likely to churn (leave the company) using Machine Learning.

## Dataset

IBM Telco Customer Churn Dataset

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## Machine Learning Workflow

1. Data Cleaning
2. Missing Value Handling
3. Feature Encoding
4. Train-Test Split
5. Logistic Regression
6. Random Forest
7. Model Evaluation

## Results

* Logistic Regression Accuracy: 78.82%
* Random Forest Accuracy: 78.82%

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix

## Key Learnings

* Data preprocessing
* Feature engineering
* Classification algorithms
* Overfitting analysis
* Model evaluation
